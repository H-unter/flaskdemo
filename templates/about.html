About This Application

This is a Flask web application that utilizes the Wikipedia API to fetch and display information. It demonstrates the power of combining Flask with external APIs to create dynamic web applications.

Additional features include:

  • Jinja HTML templates for dynamic content rendering.
  • Template inheritance for consistent layout across pages.
  • Use of HTTP POST method for form submissions.
